# financial_calc_vx# Финансовый калькулятор

Простой финансовый калькулятор для расчёта процентов и налогов.

## Функции

1. **Простые проценты** (`calculate_simple_interest`)
   - Формула: `principal * rate * time / 100`
   - Возвращает сумму простых процентов

2. **Сложные проценты** (`calculate_compound_interest`)
   - Формула: `principal * (1 + rate/(100*n))**(n*time) - principal`
   - n - количество начислений процентов в год
   - Возвращает сумму сложных процентов

3. **Налог** (`calculate_tax`)
   - Формула: `amount * tax_rate / 100`
   - Возвращает сумму налога
